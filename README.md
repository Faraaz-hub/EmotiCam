# 🚦 Traffic Sign Recognition System

## 📖 Overview
This project implements a **Traffic Sign Recognition System** using Deep Learning techniques. The goal is to accurately classify different types of traffic signs to support intelligent driving systems and road safety applications.

---

## 🚀 Key Features
- 📌 High-performance **CNN model** for image classification  
- 📌 Achieves **~98% accuracy** on test dataset  
- 📌 Image preprocessing (resizing, normalization, augmentation)  
- 📌 Multi-class classification of traffic signs  
- 📌 Deployable application using Python  

---

## 🧠 Model Architecture
- Convolutional Layers (Feature Extraction)  
- Max Pooling Layers  
- Batch Normalization  
- Dropout (Overfitting reduction)  
- Fully Connected Dense Layers  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** TensorFlow, Keras, OpenCV, NumPy, Pandas  
- **Visualization:** Matplotlib  
- **Deployment:** Streamlit / Python App  

---

## 📊 Dataset
- Contains labeled images of traffic signs  
- Includes training, validation, and test datasets  
- Preprocessed using augmentation and normalization techniques  

---

## 📈 Results
- ✅ Test Accuracy: **~98%**  
- ✅ Robust performance across multiple traffic sign classes
